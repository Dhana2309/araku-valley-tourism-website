# Araku Valley Tourism Website

This project is a tourism website designed to provide complete information about Araku Valley, a popular hill station in Andhra Pradesh. It helps tourists explore attractions, hotels, and contact details in one place.

## Features
- Home page with introduction of Araku Valley
- Attractions section displaying tourist places
- Hotels section with accommodation details
- Contact page for visitor inquiries
- Responsive and user-friendly design

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose of the Project
The purpose of this project is to build a simple tourism website that allows users to easily access travel-related information about Araku Valley.

## How It Works
Users can navigate through different sections using the menu. Each page displays useful information related to tourism in Araku Valley.
